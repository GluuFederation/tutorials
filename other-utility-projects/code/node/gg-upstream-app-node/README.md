# Node JS Demo API

## Pre-requisites

1. NodeJS > v10.x.x and NPM > 6.x.x

## Installation

1. `cd node-api-2`
1. `npm install`

## Start

```
npm start

or

node index.js
```

It will start node js server on port `3000`

## APIs

Request `GET http://localhost:3000/<anything_you_want>` to get dummy response.  

`GET`, `POST`, `DELETE` endpoints available.
